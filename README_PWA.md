# iPAS AI 考古題練習 - PWA 版本

## 檔案說明

| 檔案 | 說明 |
|------|------|
| `iPAS_AI考古題練習.html` | 主程式（包含所有題目與功能） |
| `manifest.json` | PWA 設定檔 |
| `sw.js` | Service Worker（支援離線使用） |
| `README_PWA.md` | 本說明文件 |

---

## 使用方法

### 方式一：本機直接開啟（最簡單）

1. 將此資料夾傳輸到行動裝置（USB、雲端硬碟、即時通訊軟體等）
2. 在行動裝置上開啟檔案管理器
3. 點選 `iPAS_AI考古題練習.html` 檔案
4. 選擇用瀏覽器開啟

### 方式二：安裝為 App（推薦）

#### Android 手機：
1. 用 Chrome 瀏覽器開啟 HTML 檔案
2. 點選網址列右側的「⋮」選單
3. 選擇「安裝應用程式」或「加到主畫面」
4. 桌面會出現「AI考古題」圖示
5. 之後直接點擊圖示即可開啟，無需再找檔案

#### iPhone/iPad：
1. 用 Safari 開啟 HTML 檔案
2. 點選下方分享按鈕「⎙」
3. 選擇「加入主畫面」
4. 點擊「加入」確認
5. 主畫面會出現 App 圖示

## 功能特色

- ✅ **完全離線使用** - 安裝後無需網路
- ✅ **安裝到主畫面** - 像原生 App 一樣使用
- ✅ **進度儲存** - 自動記錄答題狀態
- ✅ **多種練習模式** - 可選題數、範圍、計時
- ✅ **錯題回顧** - 檢視答錯的題目
- ✅ **響應式設計** - 完美支援手機/平板

---

## 技術說明

本應用使用 PWA (Progressive Web App) 技術：
- **Service Worker**：讓網頁可以離線運作
- **Manifest**：定義 App 名稱、圖示、顯示模式
- **Cache API**：快取所有資源供離線使用

---

## 注意事項

1. **三個檔案必須放在一起**，不能單獨移動 HTML 檔案
2. 首次開啟需要有網路或本地檔案存取權限
3. 安裝後即可完全離線使用
4. 建議定期備份此資料夾
